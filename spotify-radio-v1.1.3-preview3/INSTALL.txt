Spotify Radio for Forza Horizon 6 — install
============================================

Requires a Spotify Premium account. Free accounts will see "Get Spotify
Premium to listen" when selecting FH6 Radio — this is enforced by
Spotify for Connect devices without an official partner certificate.

Supports the PC version of Forza Horizon 6 installed via Steam, Microsoft
Store, Xbox app, or PC Game Pass.

1. Close Forza Horizon 6 if it is running.

2. Extract the contents of this ZIP directly into your Forza Horizon 6
   game directory (where forzahorizon6.exe lives), overwriting when
   prompted. Typical paths:
     C:\Program Files (x86)\Steam\steamapps\common\ForzaHorizon6\
     C:\XboxGames\Forza Horizon 6\Content\
     D:\Gaming\XboxGames\Forza Horizon 6\Content\

   If Windows Security or another antivirus blocks or quarantines the mod,
   add an exclusion for the installed version.dll file, or for the whole
   Forza Horizon 6 game directory. The mod needs this proxy DLL to start.

3. Launch the game.

4. In-game, open Settings -> Audio and set:
     Radio DJ        : Off
     Streamer Mode   : On
   Without "Streamer Mode = On" the Spotify Radio station stays hidden.
   Without "Radio DJ = Off" the in-game DJ talks over your tracks.

5. On any device on the same LAN, open Spotify, tap the device picker
   (cast icon) and choose "FH6 Radio". Play any track. The mod caches
   credentials in:
     <game>/spotify-radio/librespot-cache/credentials.dat
   so subsequent launches reconnect automatically. Spotify Connect volume
   is remembered across game launches.

6. In the game, switch the radio station until the station named
   "Spotify Radio" is selected. Spotify audio only plays through the
   in-game radio while that station is selected. The status dashboard is at:
     http://localhost:8103

Uninstall: delete version.dll and the spotify-radio/ directory from the
game folder, then use your launcher repair/verify option to restore the
patched media/Audio/* and media/UI/Textures/Anthem.zip files.
